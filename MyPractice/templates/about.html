System Development Process

STEP 1: Data Analysis

We gatherd the data from different sources available on the web like krogal. We did some review on the data as the labels or title used the number of rows and columns the data have and what information or which region is the information or data based on.

STEP 2: Data Pre-processing

For data Pre-proessing we did tokenization (helps to analyze), make lowercase (hleps to give general meaning), remove stopwords (helps in reducing noise and dimension) and stemming or lamitization (helps to normalize the words).

STEP 3: Vectorization

It is a process which helps to change the text into vectors using matrixes. We uses TF-IDF (Term frequency- Inverse document frequency) to change the text into vvector. This will set the numbers for words like give some values to random values and zero to the commonly used terms. We followed the vectorization by clasifying the data into train and test data.

STEP 4: Model Building

We uses Random forest algorithm to train the model. It is a binary decision tree that gives the output by analyzing the output from different trees and finally prioratizing the majority. Ut contains processes like bagging and feature randomness

STEP 5: Evaluating Model

We uses different methods like accuracy score testing, confustion matrx to check the error and accuracy on the system model and classification report to evaluate the working of the model.

FINISH: Data Prediction

We take the reference of the data already present in the data that we use to train the model. If the news gives the result as the label defines it i.e fake or true than we assume that the system now works perfectly else we repeat the system development from training the data or data analysis again floowing the agile development life cycle of our system.
